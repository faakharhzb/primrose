1. Have `index.md` be the default page for the site.
2. Use "markdown" library to convert markdown text to HTML.
3. Create a folder to store the output HTML files in the same structure as the input markdown files.
4. Launch a server on `http://localhost:8888` (or any other port) and host the HTML files on that server.
